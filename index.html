Demographics Classification

DEMO

This Computer Vision microservice detects for faces from an input image and predicts the demographics of each face (age, gender and race). The image is given as an input encoded using base64. The microservice will a face detector and subsequently models to predict age, gender and race. The results will then be returned back to the user as a JSON output together with bounding box coordinates for each face.Races include: Asian, Indian, White, Black, Middle Eastern and Latino Hispanic
